## Proyecto 1: Movilidad urbana y productividad economica en cuidades de Latam
Este proyecto consiste en evaluar como la movilidad urbana se relaciona con la productividad económica en las principales ciudades latinoamericanas, el objetico es identificar en qué ciudades conviene invertir en infraestructura de transporte.
Construi un dataset limpio, creé visualizaciones y entregué un reporte para la toma de decisiones en planificacion urbana e inversion en infraestructura. 

**INSTRUCCIONES**:
- Analizar la estructura, explorar, limpiar y prepar los datos para detectar inconsistencias, con lo que detectaremos cuatro columnas de tipos de datos incorrectos y realizamos una conversion corrigiendo formatos numericos y de fecha para evitar errores de calculos y comparaciones precisas.
- Renombrar columnas para facilitar la union de los dataset.
- Filtrar el año de registro para trabajar con el periodo mas reciente y relevante que es el año 2024.
- Calcular el promedio de trafico por cuidad y año, obteniendo una vision mas clara y solida para las tendencias generales.
- Unir Trafico y Economia para crear un solo DataFrame para analizar como las condiciones economicas se relacionan con la movilidad urbana utilizando clave de union llamada...
- Visualizar patrones con graficos que nos ayudan a entender como se relacionan las viaribles economicas con las de movilidad urbana.
- Trazar histogramas para cada uno de los parametros.
- Trazaremos un histograma para observar la media, mediana y detectar valores atípicos.
- Visualizar la distribución de la economía (city_gdp_capita) mediante un histograma para analizar la forma de distribucion y el valor promedio del PIB per capita.
- Camparar ambas variable trazando un un solo grafico de barras para aparezcan ambos indicadores.
