# Proyecto 1: Movilidad urbana y productividad economica en cuidades de Latam
Este proyecto consiste en evaluar como la movilidad urbana se relaciona con la productividad económica en las principales ciudades latinoamericanas, el objetico es identificar en qué ciudades conviene invertir en infraestructura de transporte.
Construi un dataset limpio, creé visualizaciones y entregué un reporte para la toma de decisiones en planificacion urbana e inversion en infraestructura.
